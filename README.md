# AriaNG-WebGUi-T-Windows

## 该项目重构中


##### 环境依赖
Visual Studio 2019
VB.net

#### 使用方法
下载Releases最新发行
下载https://github.com/mayswind/AriaNg/releases/latest 内带有AllInOne字样的版本 并解压
两个文件放入同个目录即可使用 (html文件保持index.html即可)

### 关于该项目
服务端部署了Aria2或本地Aria2 而不想打开浏览器占用大内存的用户
可以使用该软件挂至后台并日常使用
